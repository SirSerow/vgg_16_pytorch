# VGG16 with pytorch

This repository contains an implementation of VGG16 network with pytorch. 

## Folder structure

- **data**: dataset folder (initial testing with CIFAR-10)
- **doc**: documentation used in process of building this repository (articles, guides, etc)
- **src**: source code
  - *train.py*: training script
  - *predict.py*: inference script
  - **utils**: utility scripts
    - *dataloader.py*: data loader class
    - *model.py*: VGG model
